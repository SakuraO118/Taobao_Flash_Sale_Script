# 淘宝自动抢购脚本

## 功能介绍

这是一个基于 Python 和 Playwright 开发的淘宝自动抢购脚本，用于定时抢购淘宝/天猫限量发售的商品。

### 核心功能
- **定时抢购**：精确到秒的抢购时间设置
- **自动登录**：支持账号密码自动登录（可选）
- **多浏览器支持**：支持 Chrome、Firefox、Edge 浏览器
- **智能抢购**：自动点击购买按钮和提交订单
- **抢购监控**：实时监控抢购时间，确保准时执行
- **日志记录**：详细的运行日志，方便排查问题

## 技术栈

- **Python 3.7+**：主要开发语言
- **Playwright**：浏览器自动化工具，用于模拟用户操作
- **ConfigParser**：配置文件解析
- **Logging**：日志记录

## 安装步骤

### 1. 安装 Python

确保你的系统已安装 Python 3.7 或更高版本。

### 2. 安装依赖包

在项目目录下运行：

```bash
pip install -r requirements.txt
```

### 3. 安装浏览器驱动

```bash
playwright install
```

## 配置说明

编辑 `config.ini` 文件，根据你的需求进行配置：

### 账号配置

```ini
[account]
# 淘宝账号信息（可选，若不填写则需要手动登录）
username =
password =
```

### 商品配置

```ini
[product]
# 商品链接
url = https://detail.tmall.com/item.htm?id=61234567890
```

### 抢购时间配置

```ini
[flash_sale]
# 抢购时间，格式：2024-12-31 23:59:59
time = 2024-12-31 23:59:59
```

### 浏览器配置

```ini
[browser]
# 浏览器类型：chrome, firefox, edge
type = chrome
# 是否使用无头模式（不显示浏览器窗口）
headless = false
```

## 使用方法

### 步骤1：准备工作

1. **确保网络连接稳定**：建议使用有线网络，减少网络延迟
2. **关闭其他占用网络的应用**：确保抢购过程中网络畅通
3. **准备好淘宝账号**：确保账号已登录手机淘宝，方便扫码登录

### 步骤2：配置脚本

1. **打开 `config.ini` 文件**：使用记事本或其他文本编辑器打开
2. **填写商品链接**：在 `[product]` 部分的 `url` 中填写你要抢购的商品链接
3. **设置抢购时间**：在 `[flash_sale]` 部分的 `time` 中填写抢购时间，格式为 `年-月-日 时:分:秒`
4. **可选：填写账号信息**：在 `[account]` 部分填写淘宝账号和密码（不填写则需要手动登录）

### 步骤3：运行脚本

1. **打开命令提示符**：在项目目录下，按住 Shift 键并右键点击空白处，选择 "在此处打开命令窗口"
2. **运行脚本**：输入以下命令并按回车键

```bash
python main.py
```

### 步骤4：登录账号

1. **等待浏览器打开**：脚本会自动打开浏览器并跳转到淘宝登录页
2. **完成登录**：
   - 如果配置了账号密码：脚本会尝试自动登录，若失败（如需要验证码），请在浏览器中手动完成登录
   - 如果未配置账号密码：请在浏览器中手动完成登录
3. **登录完成后**：在命令提示符窗口中按回车键继续

### 步骤5：等待抢购

1. **脚本开始监控**：登录成功后，脚本会自动跳转到商品页面并开始监控抢购时间
2. **实时显示倒计时**：命令提示符窗口会显示距离抢购时间的倒计时
3. **保持窗口打开**：请勿关闭命令提示符窗口和浏览器窗口

### 步骤6：自动抢购

1. **到点自动抢购**：到达抢购时间后，脚本会自动执行抢购操作
2. **多次尝试机制**：为防止网络拥堵，脚本会多次尝试点击购买按钮和提交订单
3. **抢购结果**：抢购成功后，命令提示符窗口会显示 "抢购成功！请尽快完成支付"

### 步骤7：完成支付

1. **手动完成支付**：脚本只负责提交订单，不负责支付
2. **支付时间**：请在 24 小时内完成支付，否则订单会自动取消

## 注意事项

1. **账号安全**：
   - 本脚本仅在本地运行，不会上传你的账号信息
   - 但为了安全起见，建议使用小号或在配置文件中不填写密码，手动登录

2. **网络环境**：
   - 请确保网络连接稳定，避免在抢购过程中出现网络中断
   - 建议使用有线网络，减少网络延迟

3. **浏览器设置**：
   - 默认使用 Chrome 浏览器，如需使用其他浏览器请修改配置
   - 建议不要使用无头模式，以便观察抢购过程

4. **抢购策略**：
   - 脚本会尝试多次点击购买按钮，提高抢购成功率
   - 但由于淘宝的反爬虫机制和网络延迟，不能保证 100% 抢购成功

5. **法律合规**：
   - 本脚本仅用于个人学习和研究，请勿用于商业用途
   - 请遵守淘宝/天猫的用户协议，合理使用

## 常见问题

### 1. 脚本运行时提示 "未找到购买按钮"

- 可能原因：商品还未到抢购时间
- 解决方法：检查抢购时间设置是否正确，确保脚本在抢购开始前运行

### 2. 自动登录失败

- 可能原因：需要验证码或账号密码错误
- 解决方法：手动在浏览器中完成登录

### 3. 抢购失败

- 可能原因：商品已售罄、网络延迟或淘宝反爬虫机制
- 解决方法：
  - 提前运行脚本，确保在抢购开始前完成登录
  - 确保网络连接稳定
  - 多次尝试运行脚本

### 4. 浏览器打开后无响应

- 可能原因：浏览器驱动未正确安装
- 解决方法：重新运行 `playwright install` 命令

### 5. 脚本运行出错

- 可能原因：Python 版本过低或依赖包未正确安装
- 解决方法：
  - 确保安装了 Python 3.7 或更高版本
  - 重新运行 `pip install -r requirements.txt` 命令
